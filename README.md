# Data Lake Modeling Guide

## 📌 Sobre o projeto
Este projeto é uma revista digital interativa desenvolvida para explicar, de forma visual e estruturada, como realizar a modelagem de dados em um Data Lake.

O conteúdo aborda conceitos fundamentais como:
- Schema-on-read vs Schema-on-write
- Arquitetura Medallion (Bronze, Silver, Gold)
- Particionamento e organização de dados
- Formatos como Parquet, Delta e Iceberg
- Governança, metadados e data lineage

## 💻 Tecnologias utilizadas
- HTML
- CSS
- Design responsivo
- Estrutura modular de layout

## 🎯 Objetivo
Demonstrar, de forma prática, conceitos de engenharia de dados e organização de Data Lakes, unindo conhecimento técnico com apresentação visual.
